1. RICH TEXT EDITOR IMPROVEMENTS
=================================

Default Mode: Plain Text
-------------------------
- Modal compose default plain text biasa
- Rich text hanya muncul saat klik tombol "T"
- Mode reset ke plain text saat modal ditutup
- Better UX untuk user yang tidak butuh formatting

Toolbar Optimization
--------------------
- Ubah dari flex ke inline-flex (compact)
- Icon size: 3.5x3.5 (dari 4x4)
- Button size: 7x7 (dari 8x8)
- Padding: 1.5 (dari 2)
- Spacing: 1 (dari 1)
- Divider: 5px height (dari 6px)

Features Removed
----------------
- ❌ Numbered list (1. 2. 3.)
- ❌ Link button (link auto-detect)
- ❌ Format hint text

Features Kept
-------------
- ✅ Bold (**text**)
- ✅ Italic (*text*)
- ✅ Underline (<u>text</u>)
- ✅ Bullet list (- item)
- ✅ Auto-continue list on Enter

Markdown Parser Updates
-----------------------
- Fix bold & italic regex conflict
- Remove numbered list parsing
- Simplified bullet list only
- Proper <ul> wrapping
- Better HTML output
