BUG FIXES

1. FIX TEMA KEMBALI CERAH SAAT REFRESH

Memperbaiki bug kritis dimana tema dark mode kembali ke light mode setiap kali halaman di-refresh atau reload.

Masalah Sebelumnya:

  • User memilih dark mode
  • Setelah refresh halaman, tema kembali ke light mode
  • User harus set ulang dark mode setiap kali buka aplikasi
  • Preferensi tema tidak tersimpan dengan benar

Solusi:

  • Implementasi proper theme persistence di localStorage
  • Theme initialization saat app startup
  • Sync theme state dengan localStorage
  • Prevent theme flash on page load

Impact:

User experience lebih baik, tema konsisten setelah refresh, tidak perlu set ulang preferensi tema

2. FIX UPLOAD POSTINGAN DI HALAMAN LAIN

Memperbaiki bug dimana upload postingan hanya bisa dilakukan di halaman Home, tidak bisa di halaman lainnya seperti Profile, Bookmarks, dll.

Masalah Sebelumnya:

  • Upload button hanya muncul di Home page
  • User harus kembali ke Home untuk upload post
  • Workflow tidak efisien
  • Inconsistent user experience

Solusi:

  • Global upload functionality
  • Upload button accessible dari semua halaman
  • Consistent post creation flow
  • Better component architecture

Impact:

Upload post dari halaman manapun, workflow lebih efisien, better user experience

FITUR BARU

FITUR "CERITAMU" (STORIES)

Halaman khusus untuk menampung cerita seru atau curhatan dalam bentuk tulisan panjang. Fitur ini bisa digunakan sebagai tempat draft sebelum diunggah ke platform cerpen atau lainnya.

Fitur Detail:

  • Editor lengkap untuk menulis cerita panjang
  • Support rich text formatting
  • Auto-save untuk prevent data loss
  • Kategori cerita (Senang, Sedih, Lucu, Horor, dll)
  • Character count dan reading time estimation
  • Draft management
  • Search dan filter cerita
  • Export cerita ke file
  • Dark/Light mode support

Use Cases:

  • Menulis cerita pendek atau novel
  • Draft untuk platform cerpen (Wattpad, Fizzo, dll)
  • Personal journaling dengan format panjang
  • Curhatan atau refleksi pribadi
  • Creative writing practice

Keunggulan:

  • Offline-first: Semua cerita tersimpan lokal
  • Privacy: Tidak ada yang bisa baca kecuali kamu
  • No limits: Tulis sepanjang yang kamu mau
  • Auto-save: Tidak khawatir kehilangan tulisan

PERUBAHAN DESAIN

MAJOR UI REFRESH - GLASSMORPHISM

Merapikan dan memperbarui desain yang sebelumnya terlihat flat menjadi lebih modern dengan konsep glassmorphism serta sudut rounded yang lebih besar.

1. Glassmorphism Effect

  • Backdrop blur untuk card dan modal
  • Semi-transparent backgrounds
  • Layered depth dengan shadows
  • Modern dan elegant look

2. Rounded Corners

  • Border radius lebih besar (8px → 16px)
  • Consistent rounded corners
  • Softer appearance
  • More friendly UI

3. Card Design

  • Glass effect dengan blur
  • Subtle shadows untuk depth
  • Better spacing dan padding
  • Improved readability

4. Navigation

  • Glassmorphism navbar
  • Floating effect
  • Better visual hierarchy
  • Smooth transitions

Impact:

More modern appearance, better visual hierarchy, improved user experience, professional look and feel, better brand identity
